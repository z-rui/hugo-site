---

XeTeX 是 TeX 排版系统的扩展，增加了对 Unicode 和 OpenType(R) 的支持。因此可被用于中文排版。

我写了 [typezh](https://github.com/z-rui/typezh)，用于在 Plain TeX 的基础上编写中文文档。可以参考其中的 `readme.tex` 了解其用法。或者，更好地，使用
```
xetex readme.tex
```
将说明文件编译成 [PDF](/media/typezh.pdf)，然后查看。
